# Phototeque (local)

Simple projet où l'on upload des photos sur un site.

## Prérequis

- NodeJS >= v20
- Mongodb >= v5

## Installation

Clôner le dépôt git. Ensuite faire les commandes suivantes :

```
npm init
```
```
node index.js
```

## Technologies utilisées

- NodeJs
- ExpressJS / HTML
- Bootstrap
- Mongodb
